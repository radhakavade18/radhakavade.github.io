Poll Maker

The aim of this project is to creating and sharing the polls with images, text, question & answers and sharing thoughts with the voice messages in a very easy way.

Project Idea

While creating the poll we can add people and set our poll to private and public. That can specify that our poll is going to shared as public or private with the specific people or group who has added while creating the poll

Target Audience

I have being seeing many of my friends worry as what to eat for lunch, what to buy or select from a list. Unlike other social networks, all the posts are in the form of voting.

  • The highlight of this project is mainly focussed to give the best user experience in terms of using the application easily and effectively.
  • Create & share the poll with selective people/friends
  • Making user more interactive and connect with friends for more clear suggestions with more specific opinions.
  • Add people/friends in one place to get the fastest voting results.

01. Onboarding

PollMaker does not display the images in the post section at first. The benefit of not having an image is that the user can see a lot of content on the screen. On the other hand if there is no image, user can pass by without reading the content. So I added the bold title of the post. Thus, user doesn’t need to read all the text on the post, they can only read the title and get to know the content of the post.

02. Register User
