Website Portal Design

Rental Property Portal

Project Discription

  • The goal of the project is to book property on rent in Australia. It is a platform where four main portals are worked together to get rented house to the tenant.
  • Admin Is created by the property manager. Property managers create property owners. The owner's property is managed by PM. Property managers send application forms for property applicants.
  • Applicant can fill the form and send back to PM. Property managers send the reference application in summery to the owner and the owner will select the applicant for further process.
  • After verification of documents and all details provided by applicant the prescribed form is generated. After this process applicant becomes a tenant.

My Approach

  • Gathering and evaluating user requirements, in collaboration with product managers and team.
  • Illustrating design ideas using storyboards, process flows and sitemaps. Designing graphic user interface elements, like menus, tabs and widgets.
  • Develop UI mockups and prototypes that clearly illustrate how sites function and look like.
  • Work with other developers to ensure that the website is functional and visually appealing.
  • Ensure that the website is optimized for various devices. Identify and troubleshoot UX problems (e.g. responsiveness)
